# Bear Brewers Beverage 🍺

A modern and responsive beverage brand website built using **React** and **Vite**. The project focuses on creating a visually engaging landing page with smooth animations, clean UI, and a responsive design.

## 🚀 Live Demo

**Website:** https://bear-brewer-beverages.vercel.app/

## 🛠️ Tech Stack

- React
- Vite
- JavaScript
- CSS

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Madhav-Agg/Bear-Brewers-Beverage.git
```

Navigate to the project folder:

```bash
cd Bear-Brewers-Beverage
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
